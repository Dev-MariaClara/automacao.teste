# 📊 Automação de Contas a Pagar - Olist

Um projeto desenvolvido em **JavaScript** para automatizar a extração e consolidação de dados de contas a pagar do sistema Olist. A ferramenta simplifica a rotina financeira: através do clique em um link, os dados são processados e um arquivo Excel é gerado automaticamente com o relatório diário.

##  Funcionalidades
- **Geração de Relatórios em Excel:** Exportação diária de dados financeiros consolidados, formatados e prontos para uso.
- **Automação de Processos:** Elimina o trabalho manual de coleta e tabulação de dados no Olist.
- **Acionamento Simples:** A execução de toda a rotina de automação é disparada de forma intuitiva, apenas acessando o link configurado.

## Tecnologias Utilizadas
- **JavaScript:** Lógica principal de automação, manipulação das informações e exportação do arquivo.
- **Vercel:** Utilizado para realizar o deploy da aplicação e gerar o link web de acesso direto.
- **Git & GitHub:** Controle de versão e gestão do repositório.

##  Como Executar

1. Clone este repositório para a sua máquina local:
   ```bash
   git clone https://github.com/seu-usuario/nome-do-repositorio.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd nome-do-repositorio
   ```
3. Para testar o script localmente (certifique-se de ter o Node.js instalado), execute:
   ```bash
   node nome-do-arquivo.js
   ```
   *Nota: A versão de produção pode ser executada diretamente acessando o link gerado pelo deploy na Vercel.*
